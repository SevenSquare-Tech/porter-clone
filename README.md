# Porter Clone (Flutter App)

## Description

Porter Clone is a Flutter-based mobile application designed to replicate core functionalities of a logistics/service booking platform.

[A Complete Guide to Create Porter Clone in Flutter.](https://www.sevensquaretech.com/build-porter-clone-flutter-with-free-code/)

## Version

1.0.0+1

## Environment

Dart SDK: >=2.17.6 <3.0.0

## Features

- User Authentication (Firebase)
- Location Services (Geolocator & Google Maps)
- Booking & Tracking
- Payment Integration (Paystack)
- File Uploads & Media Handling
- Charts & Analytics
- State Management (Bloc & Provider)

## Dependencies

- flutter_bloc
- firebase_auth
- google_maps_flutter
- geolocator
- http
- shared_preferences
- image_picker
- audioplayers
- file_picker

## Installation

1. Clone the repository:
   git clone https://github.com/SevenSquare-Tech/porter-clone

2. Navigate to project:
   cd porter-clone

3. Install dependencies:
   flutter pub get

4. Run the app:
   flutter run

## Assets

All assets are located in the /assets directory.

## Notes

- Ensure proper API keys for Google Maps & Firebase are configured.
- Update environment configurations before running in production.
